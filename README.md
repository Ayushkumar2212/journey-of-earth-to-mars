# 🚀 Planet Mars: The Interstellar Journey

A high-fidelity, interactive web experience documenting the voyage from Earth to the Red Planet. This project features stunning **GSAP-powered animations**, **ScrollTrigger** effects, and a modern **glassmorphism** UI.



## 🌌 Features
- **Cinematic Intro**: Watch the rocket take off as you land on the page.
- **Scroll-Linked Animations**: A progress-tracked rocket moves as you explore the journey.
- **8 Dynamic Phases**: From Earth Departure to Surface Touchdown, each phase features unique scroll-revealed content.
- **Futuristic UI**: Clean, glassmorphism-based design with an Orbitron & Space Grotesk font pairing.
- **Smooth Performance**: Powered by **Vite** for lightning-fast development and optimized production builds.

## 🛠️ Tech Stack
- **Framework**: [Vite](https://vitejs.dev/)
- **Animation**: [GSAP](https://gsap.com/) (GreenSock Animation Platform)
- **Styling**: Vanilla CSS (Modern CSS3 with Variables)
- **Structure**: Semantic HTML5

## 🚀 Getting Started

### 1. Prerequisites
Ensure you have [Node.js](https://nodejs.org/) installed on your machine.

### 2. Installation
Clone the repository (or extract the folder) and navigate into it:
```bash
# Navigate to the project folder
cd iitp

# Install dependencies
npm install
```

### 3. Local Development
To run the project locally with hot-reloading:
```bash
npm run dev
```
Open [http://localhost:5173/](http://localhost:5173/) in your browser.

## 🏗️ Building for Production
To create an optimized production build:
```bash
npm run build
```
The site will be generated in the `dist/` folder.

## 🌍 Deployment

### **Vercel / Netlify**
Simply connect your GitHub repository to [Vercel](https://vercel.com) or [Netlify](https://www.netlify.com). They will auto-detect the Vite configuration and deploy it for you.

Alternatively, run:
```bash
# For a quick manual deploy (if you have the CLI)
npx vercel --prod
```
